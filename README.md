# منصة المراجعة الداخلية والرقابة بالسجون

نسخة مطورة مبنية على مشروع `audit_platform_integrated_latest` مع تضمين التعديلات الرئيسة التالية:

- فصل دور **مختص الإدارة الرئيسية** عن **مدير إدارة المراجعة الداخلية**
- دعم الإسناد من **الجهاز الرئيسي** أو **المنطقة** أو **إسناد مختلط**
- تحسين الواجهة إلى **Side Menu يمين** وهوية حكومية أكثر وضوحًا
- إضافة شعاريّْن بصيغة SVG كعناصر واجهة قابلة للاستبدال بملفاتكم الرسمية
- توسيع الداشبوردات ورسومات KPI/KRI لكل دور
- دعم سجلات تتبع **Audit Log** وعرضها داخل صفحة المهمة
- تحسين نماذج التنفيذ والملاحظات والحقول الشرطية
- تحديث بيانات seed لتكون أقرب للواقع التشغيلي

## التشغيل

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python run.py
```

ثم افتح:

```text
http://127.0.0.1:5000/login
```
